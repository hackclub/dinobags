# Dinobags

Dinobags is a simple HCB bot that uses the V4 API to automatically top up HCB grants. Very useful for large programs that automate sending a lotta grants.

## Usage
1. Invite `dinobags@hackclub.com` to your HCB org as a manager.
2. Have people who want grant topups add their grant URL in the message! (Storing grant data in your db? Consider pre-filling the donation form like so: `https://hcb.hackclub.com/donations/start/flavortown?email=amber@hackclub.com&message=Top%20up%20for%20https://hcb.hackclub.com/grants/nZIQ6K&name=Amber%20Barnes&goods=true`)
3. That's it! Within a couple of minutes the grant should be automatically topped up
